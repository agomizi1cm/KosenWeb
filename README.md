
# 高専生活特化WEBアプリ

## アプリ概要

本アプリケーションは，年間・週間・科目単位で情報を管理・閲覧する Web アプリケーションです．
利用者はトップページから各機能画面に遷移し，データの閲覧および一部編集を行うことができます．

Node.js を用いたサーバサイド処理と，EJS によるテンプレートエンジンを使用して実装しています．

## 動作環境

以下の環境で動作確認を行っています．

- OS: Windows 11 / macOS / Linux
- Node.js: v24.x+
- npm: 11.x+
- ブラウザ: Google Chrome（最新版推奨）

## セットアップ方法

以下のコマンドを実行し，必要なパッケージをインストールします．

```bash
npm install
```

`package-lock.json`を同梱しているため，基本的には環境差異なくインストール可能です．

## 実行方法

```bash
node index.js
```

起動後，ブラウザで以下のURLにアクセスしてください．

http://localhost:3000/

使用ポートは`index.js`内で指定されています．

## アプリの使用方法

1. ブラウザでトップページ（index画面）が表示されます．
1. 画面上のアイコンから以下の画面に遷移できます．
    - 年間予定表
    - 授業時間割
    - テスト日程表
    - 図書館OPAC
    - 管理者ページ
1. 各画面では，登録済みデータの閲覧および操作が可能です．
